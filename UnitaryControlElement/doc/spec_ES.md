Descripción global: **Un elemento de control unitario combina varios componentes de control en un solo producto, como un termostato o un humidostato.  Un elemento de control unitario proporciona un alojamiento para una agregación de elementos de control o de distribución eléctrica que, combinados, realizan un propósito singular (unitario). Cada elemento de la agregación puede tener su propia representación geométrica y ubicación.**  
versión: 0.0.1
